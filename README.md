# ☕ Atividades de Java — SENAI

Este repositório foi criado com o objetivo de armazenar e organizar as atividades desenvolvidas durante as aulas de Java no SENAI.

## 📚 Conteúdos estudados

Ao longo das atividades, são abordados conceitos fundamentais e intermediários da linguagem Java, incluindo:

- Lógica de programação
- Sintaxe básica do Java
- Estruturas condicionais e de repetição
- Métodos e funções
- Vetores e matrizes
- Programação Orientada a Objetos (POO)
  - Classes e objetos
  - Encapsulamento
  - Herança
  - Polimorfismo
  - Abstração
- Tratamento de exceções
- Coleções
- Boas práticas de programação

## 🎯 Objetivo do repositório

O principal objetivo deste repositório é:

- Registrar a evolução do aprendizado em Java;
- Armazenar exercícios e projetos feitos em sala;
- Facilitar consultas futuras e revisões de conteúdo;
- Compartilhar conhecimentos e práticas de programação.

## 🛠️ Tecnologias utilizadas

- Java
- IDEs como IntelliJ IDEA / Eclipse / VS Code
- Git e GitHub

## 📂 Organização

As atividades estão separadas por temas, aulas ou exercícios, facilitando a navegação e o acompanhamento do conteúdo estudado.

## 🚀 Observações

Este repositório possui fins educacionais e representa minha jornada de aprendizado em desenvolvimento Java no SENAI.
